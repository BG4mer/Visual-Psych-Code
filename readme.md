# FNF Mod Editor

A **VS Code–like editor for Friday Night Funkin' mods** that runs directly in your browser. Now with **top tabs** for multiple files, Lua/Haxe coding, snippets, and live Lua execution with **Fengari**.

## Features

- Lua & Haxe syntax highlighting using Monaco Editor  
- **Top Tabs** like VS Code with close buttons  
- Multiple files (switchable in tabs)  
- Insert code snippets for FNF modding  
- Create new files (Lua or Haxe)  
- Save current file locally  
- Run Lua scripts in-browser with Fengari  
- Fully static → works with **GitHub Pages**  

## How to Use

1. Clone or download this repository.  
2. Open `index.html` in a modern browser, or host it using GitHub Pages:  
   - Go to your repository → **Settings → Pages**  
   - Set **Branch** to `main` and **Folder** to `/root`  
   - Click **Save**  
3. Use **top tabs** to switch between files.  
4. Use the sidebar for snippets, new files, save, or run Lua.
